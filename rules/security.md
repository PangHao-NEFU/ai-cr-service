# 安全规范（始终加载）

## 认证与授权
- 禁止硬编码密码、API Key、Token 等敏感信息
- 使用环境变量或密钥管理服务存储敏感信息
- 所有 API 接口必须有权限验证

## 数据安全
- SQL 查询必须使用参数化，禁止字符串拼接
- 禁止在前端代码中暴露敏感数据
- 用户输入必须校验和转义

## 常见漏洞防范
- SQL 注入：使用参数化查询或 ORM
- XSS：输出时进行 HTML 转义
- CSRF：关键操作需要 CSRF Token
- 路径遍历：校验文件路径，禁止 `..` 等危险字符

## 日志与监控
- 禁止在日志中输出敏感信息（密码、Token）
- 关键操作必须有审计日志
